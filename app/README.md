# MicroGPT Education Scroll Web

React + Tailwind + GSAP 기반의 교육용 풀스크린 스크롤 페이지입니다.

## Stack

- React 19
- Vite 7
- Tailwind CSS 4 (`@tailwindcss/vite`)
- GSAP + ScrollTrigger

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
